# WaterWise Assistant Chatbot

## SDG
SDG 6 – Clean Water and Sanitation

## Project Description
This project is a WaterWise Assistant Chatbot developed using Botpress. The chatbot helps users learn about clean water, sanitation, hygiene, water conservation, and water pollution.

## Features
- Clean water awareness
- Sanitation education
- Water conservation tips
- Water pollution awareness
- SDG 6 information

## Tools Used
- Botpress
- Generative AI
- GitHub

## Deployment
Bot deployed using Botpress Cloud.
